# PATIENT MANAGEMENT SYSTEM

This project is a Patient Management System API built with FastAPI to demonstrate real-world backend development concepts such as data validation, CRUD operations, computed health metrics, sorting, and structured error handling.

The application stores patient data in a JSON file, making it lightweight and ideal for learning REST API architecture and backend logic without database complexity.

# Key Features

Full CRUD operations for patient records

Strong data validation using Pydantic models

Automatic BMI calculation and health verdict generation

Sorting functionality by height, weight, or BMI (ascending/descending)

Clean RESTful API structure with proper HTTP status codes

Lightweight JSON-based data persistence

Interactive Swagger/OpenAPI documentation
